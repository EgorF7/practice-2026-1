# О ПРОЕКТЕ

## ОПИСАНИЕ ПРОЕКТА

Проект ***Project Spacewolf*** - полнофункциональная демо-версия трехмерной компьютерной игры в жанре космического шутера с комбинацией привычных для данного жанра систем передвижения - системы с прямолинейным движением по локациям, имеющим определенную степень свободы; уникальным визуальным стилем и механикой смены управляемого корабля.

## ЦЕЛЬ ПРОЕКТА

Целью проекта является создание **демо-версии** игры жанра космический шутер с возможностью смены управляемого игроком космического корабля.

## ЗАДАЧИ ПРОЕКТА

- Проведение анализа актуальности проекта.
- Разработка и утверждение единой художественной концепции.
- Создание механики полета по заданному маршруту с возможностью переключения между играбельными кораблями.
- Реализация битвы с боссом.
- Построение игрового тестового уровня.
- Наполнение демо-версии визуальными и звуковыми элементами для достижения максимальной презентабельности.

## АКТУАЛЬНОСТЬ

Актуальность проекта заключается в создании продукта, космического шутера, преодолевающего клише своего жанра через инновационные проектирование уровней и уникальную визуальную подачу.
